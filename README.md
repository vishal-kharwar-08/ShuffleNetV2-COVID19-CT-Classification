# COVID-19 CT Scan Classification using ShuffleNet V2

## Project Overview
This project implements the ShuffleNet V2 deep learning architecture using PyTorch for COVID-19 CT scan image classification.

The model classifies CT scan images into:
- Covid
- Healthy
- Others
